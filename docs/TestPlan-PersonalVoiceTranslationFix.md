# テスト計画書: Personal Voice 翻訳機能修正の検証

**日付**: 2025年12月4日  
**バージョン**: 1.0.3  
**テスト対象**: Personal Voice 使用時の翻訳機能の修正  
**関連Issue**: Personal Voice を使うときに翻訳が行われていない

---

## テストの目的

Personal Voice の Speaker Profile ID を設定した状態で翻訳先言語を変更しても、正しく翻訳が実行されることを確認する。

---

## テスト環境

### 必須条件
- Azure Speech Service のサブスクリプションキー
- Azure Speech Service のリージョン（例: japaneast）
- Personal Voice の Speaker Profile ID（推奨）
- マイクが利用可能なPC
- 対応ブラウザ（Chrome 90+, Edge 90+, Safari 14+, Firefox 88+）

### テスト用言語ペア
1. **日本語 → 英語**（主要テスト）
2. **日本語 → 中国語**
3. **英語 → 日本語**
4. **英語 → フランス語**

---

## テストケース

### TC001: 初期設定と基本的な翻訳動作の確認

**目的**: Personal Voice を使用した基本的な翻訳が正常に動作することを確認

**前提条件**:
- アプリケーションを初めて起動
- 設定が未保存の状態

**手順**:
1. ブラウザで `src/index.html` を開く
2. 設定ボタン（⚙️）をクリック
3. 以下を入力:
   - Subscription Key: [あなたのキー]
   - Service Region: [あなたのリージョン]
   - Personal Voice ID: [あなたの Speaker Profile ID]
   - 認識元言語: 日本語 (ja-JP)
4. 「保存」をクリック
5. 翻訳先言語を「英語 (en-US)」に設定
6. 「翻訳を開始」ボタンをクリック
7. マイク権限を許可
8. 日本語で話す: "こんにちは、今日はいい天気ですね"

**期待される結果**:
- ✅ 設定が保存される
- ✅ サービスが初期化される（ステータス: "準備完了"）
- ✅ マイク権限の要求が表示される
- ✅ 「翻訳を開始」ボタンが有効になる
- ✅ 音声入力が開始される（ステータス: "翻訳中..."）
- ✅ 日本語の原文が左側に表示される
- ✅ 英語の翻訳が右側に表示される: "Hello, it's a nice day today"
- ✅ Personal Voice で英語の音声が再生される
- ✅ コンソールに以下のログが表示される:
  ```
  [SpeechRecognition] 翻訳先言語: en
  [SpeechRecognition] 認識完了: TranslatedSpeech
  [SpeechSynthesis] Personal Voice 用 SSML - ベース音声: DragonLatestNeural
  ```

## 追加の確認項目

### コンソールログの確認

各テストケースで、以下のログが適切なタイミングで出力されることを確認してください：

**言語変更時**:
```
[AppController] 翻訳先言語変更: [言語コード]
[AppController] 翻訳先言語変更により音声認識・合成サービスを再初期化
[AppController] 翻訳中のため、先に停止します (翻訳中の場合のみ)
[SpeechRecognition] 認識停止 (翻訳中の場合のみ)
[SpeechRecognition] 初期化開始
[SpeechRecognition] 翻訳先言語: [言語コード(短縮形)]
[SpeechSynthesis] 初期化開始
[SpeechSynthesis] Personal Voice モード - ベース音声を使用: DragonLatestNeural (Personal Voice使用時)
または
[SpeechSynthesis] 標準音声モード (標準音声使用時)
```

**翻訳実行時**:
```
[SpeechRecognition] 認識完了: TranslatedSpeech
[SpeechRecognition] 最終原文: [原文]
[SpeechRecognition] 最終翻訳: [翻訳]
[SpeechSynthesis] Personal Voice 用 SSML - ベース音声: DragonLatestNeural
```

**エラー発生時**:
```
[AppController] サービス再初期化エラー: [エラーメッセージ]
[AppController] 音声認識サービスのクリーンアップエラー: [エラー] (エラー時のみ)
[AppController] 音声合成サービスのクリーンアップエラー: [エラー] (エラー時のみ)
```

### UIの確認

各テストケースで、以下のUI要素が正しく更新されることを確認してください：

**ステータス表示**:
- 準備中... (初期化前)
- 準備完了 (初期化完了後、翻訳停止中)
- 翻訳中... (翻訳実行中)
- 再初期化エラー (エラー発生時)

**開始/停止ボタン**:
- 無効 (初期化前)
- 有効 (初期化完了後)
- テキストが「翻訳を開始」⇔「停止」に切り替わる

**チャット表示エリア**:
- 原文が左側（青系統の吹き出し）に表示される
- 翻訳が右側（緑系統の吹き出し）に表示される
- 自動スクロールが動作する
